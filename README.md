# ⚔️ Promisance Reborn

A modernized React reimagining of the classic QM Promisance turn-based empire strategy game.

## Features

- **12 playable races** — Human, Elf, Dwarf, Troll, Orc, Drow, Vampire, Viking, Minotaur, Warlock, Golem, Pixie — each with unique stat modifiers and lore
- **Full empire management** — explore land, collect taxes, farm food, build structures, recruit military, cast spells
- **Realistic canvas-rendered torches** with particle flame systems, ember effects, and ambient light bloom
- **Cinematic turn effects** — full-screen visual feedback on every action
- **Rich race identity system** — animated portraits, lore text, trait pills, and stat bars
- **Glass-morphism UI** with dark purple/gold color scheme, Cinzel typography, and animated particle background

## Tech Stack

- React 18
- HTML5 Canvas (torches, particle field)
- CSS animations & keyframes
- No external UI libraries

## Getting Started

```bash
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000).

## Build for Production

```bash
npm run build
```

## Original Game

Based on [QM Promisance](https://github.com/qmtechnologies/promisance) by QMT Productions — a PHP turn-based browser strategy game originally released in the early 2000s.
